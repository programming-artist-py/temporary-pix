What is Pixy Dust?

Pixy Dust is a decentralized web ecosystem designed to empower anyone to create, register, and serve websites without needing a traditional centralized server. It combines a custom DNS system, a registrar API, and seamless GitHub integration to make hosting easy and accessible.

PixDNS - The Custom Domain Name System

PixDNS is a lightweight DNS server tailored for the .pix top-level domain. It resolves .pix domains to IP addresses controlled by users, enabling fast and flexible domain name resolution within the Pixy Dust network.

Pix Registrar - Your Domain Management API

The Pix Registrar is an HTTP API that lets you register .pix domains by linking them to your server IP, port, and a GitHub repository containing your website content. Once registered, your site is automatically cloned and served within the network.

Register your site by sending a POST request with your domain, IP, port, and repository URL to the registrar API.
